# Checkpoints

Trained checkpoints are not released in this repository.

Use this directory only for local experiments. Keep model weights outside Git, for example:

- `checkpoints/coduar_local.pth`
- `/path/to/private/checkpoints/coduar_checkpoint.pth`

The example configs refer to local checkpoint paths that users should fill themselves.

Do not commit `.pth`, `.pt`, `.ckpt`, or other trained model files.
